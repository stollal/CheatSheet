Welcome to my HTML & CSS Cheat Sheet

This is a quick guide of basic concepts for reference.

HTML Tag/Attribute Purpose Additional Notes
div helps to section html code this is good to have for clean look and easy readability and makes it easier to add class or id attributes and helps with styling
span helps to make specific changes to inline elements I mostly use this if trying to make specific text in a paragraph stand out
id helps to make unique identifier for a specific section in html this can be a big help when it comes to css styling
class helps to make unique identifier for a group of sections in html this can be a big help when it comes to css styling and you want all things in those sections to be uniform
h1 - h6 there are 6 different sizes of headings, h1 - h6 great for titles, section headers, photo captions, and much more
p the paragraph tag is where all the main body of text would go in html this tag is one of the most often used
img image tag is used to insert an image this tag doesn't need an opening and closing tag it is one tag that you close and height and width attributes
video this tag is so you can add videos and video controls This uses an opening and closing tag with the source tag in between
audio this tag is so you can add audio and audio controls to your webpage this also has an opening and closing tag but the src and controls go in the opening tag
figure an image, illustration, diagram, code snippet, etc., that is referenced in the main flow of a document, but that can be moved to another part of the document or to an appendix without affecting the main flow represents self-contained content, potentially with an optional caption, which is specified using the figcaption element
figcaption used to caption the figure or add a legend to the figure element used within the figure opening and closing tags
strong use these tags around text you want to be bold this is used to help emphasize text
link this is used in the head tags to link stylesheets or other items you want to be available to your html file I mainly use this for css style sheets or adding new web fonts
body this tag contains the main content of your html code that will show on your webpage
head this is where you put the metadata for your webpage includes things title, links, etc.
title this is the title that will display on tab in an internet browser
table this is tag that would start a table like is currently displaying this has child elements of thead, th, tbody, tr, td, etc.
br this is a break line to add white space to your webpage this is only one tag not an opening and closing tag


CSS Element Purpose Additional Notes
inline styling this is styling added to the html file inline with what ever area you are trying to change the display to this can get unorganized and hard to keep clean
styling sheet this is a css file that is used to organize all styling elements for any specified code elements this is the preferred and recommended route
box model every box is composed of four parts, defined by their respective edges: the content edge, padding edge, border edge, and margin edge this is the general layout of every element displayed on a webpage and good to keep in mind when creating your css
